# 📸 Photography Portfolio Website

A modern and responsive photography portfolio website designed to showcase stunning visual stories, creative photography projects, and professional photography services.

## 🌐 Live Demo

Visit the website:

https://gajanansagadevan.github.io/Photography/

## 📖 About the Project

This photography portfolio website was created to present photographic work in a visually appealing and professional manner. The site highlights different photography categories, featured galleries, and creative projects while providing visitors with an immersive viewing experience.

The goal of this project is to:

* Showcase photography skills and creative vision
* Display high-quality photo collections
* Provide a professional online presence
* Offer an easy-to-navigate user experience
* Demonstrate front-end web development skills

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and clean user interface
* Interactive image galleries
* Smooth navigation
* Professional portfolio presentation
* Fast loading performance
* Elegant photography-focused layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
